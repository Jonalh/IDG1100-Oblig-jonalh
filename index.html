2. What is user space?

  • All code that runs outside the kernel.
  • Software and programs the user can interact with.

3. What is the shell?

  • User interface that is used to access the operating system's services.
  • Command Line Interface (GLI), and Graphical User Interface (GUI).
  • When using GLI, you are working with commands and scripting languages.

4. Where is the shell located?

  • The shell is located in the User Space, because the user can see and interact with the shell.

5. What is the main task of the shell?

  • Manipulating data
  • Working with the operating system's services

24. Create a step by step tutorial on how to create a cronjob that prints “Task scheduled at” + DATE. Cronjob is scheduled every 30 minutes on today’s date (only today).

  • Write crontab -e to edit the cron job
  • Write 30 * 16 11 * (todays date), it will execute every minute 30 on this day only
  • After that write: echo "Task scheduled at $(date)"
  • Assign it to a txt file: >> /home/example1.txt
  • The line will look like this: 30 * 16 11 * echo "Task scheduled at $(date)" >> /home/example1.txt
  • Navigate to /home, and touch a file named example1.txt
  • Start the cron service: service cron start
  • Check the cron status: service cron status
  • Cat the example1.txt file to view the output
